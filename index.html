Privacy

If you're reading this, it's because you want to know: (1) what data we may be logging; (2) why we are logging this; (3) security of your own information. In order to make improvements and provide the best possible experience, this game anonymously records user interactions and IP addresses. No personal information is recorded. If you still have questions or concerns after reading this, please contact us at eland@cs.cornell.edu

Information We Track

In this game, we actively log information such as:

  • Playtime
  • Death Locations
  • Enemy Deaths
  • Mute/Unmute
  • and other game related data...
We do not log any personally-identifiable information. Each session is given a random UUID.

How We Use This Information

We are currently using the data as part of our course at Cornell University, CS4154: Analytics-Driven Game Design. We generate heatmaps (of player deaths), burndown charts, and state transition diagrams. These logistics help us improve our game and find ways to make the game less frustrating for players. No information gathered via this game will be shared or made public outside of the course.
